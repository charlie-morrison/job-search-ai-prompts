## Career Research

### 1. Career Direction Finder
```
I have [X years] experience in [current field]. My key skills are [skill 1, skill 2, skill 3].
I enjoy [what you enjoy] and want to earn [target salary]. Suggest 5 career paths that match
my profile, including emerging roles I might not have considered. For each, explain: required
skills gap, typical salary range, demand outlook for 2026-2027, and a 90-day transition plan.
```

### 2. Company Deep Dive
```
I'm preparing to apply at [company name]. Research this company and provide: their mission and
values, recent news or product launches, company culture indicators, interview process reputation,
and 3 things that would impress a hiring manager there. Also suggest how my background in
[your field] aligns with their needs.
```

### 3. Job Title Mapping
```
I currently work as a [current title]. What other job titles should I search for that require
similar skills? Include: exact title variations, related roles I could transition to, titles
used by startups vs corporations for the same role, and any emerging titles in this space.
```

## Resume Optimization

### 4. Resume Rewrite for ATS
```
Here's my resume: [paste resume]. Rewrite it to be ATS-friendly for this job posting:
[paste job description]. Incorporate relevant keywords naturally. Use strong action verbs.
Quantify achievements where possible. Keep it to 1-2 pages. Format with clear section
headers: Summary, Experience, Skills, Education.
```

### 5. Achievement Quantifier
```
I worked as a [role] at [company] for [time]. My responsibilities included [list tasks].
Help me turn these responsibilities into quantified achievements using the formula:
"[Action verb] + [what you did] + [measurable result]". If I don't have exact numbers,
suggest reasonable estimates I could verify or ask "what percentage/amount would you
estimate for X?"
```

### 6. Skills Gap Analyzer
```
Here's a job description I'm interested in: [paste JD]. And here's my current resume:
[paste resume]. Identify the gaps between what they want and what I have. For each gap,
suggest: how critical it is (deal-breaker vs nice-to-have), how to address it quickly
(courses, projects, certifications), and how to frame existing experience to partially cover it.
```

## Cover Letters

### 7. Tailored Cover Letter
```
Write a cover letter for this position: [paste job description]. My relevant experience:
[brief summary]. My key achievements: [list 2-3]. Tone: professional but personable.
Structure: hook (why this company), value (what I bring), proof (specific examples),
close (enthusiasm + call to action). Keep it under 350 words.
```

### 8. Career Change Cover Letter
```
I'm transitioning from [current role/industry] to [target role/industry]. Write a cover letter
that: acknowledges the transition honestly, connects my transferable skills to the new role,
shows genuine passion for the new field, and addresses why I'm making the change. Avoid
clichés like "I'm a fast learner." Use specific examples from my background: [brief summary].
```

## Interview Prep

### 9. STAR Story Generator
```
I need to prepare STAR stories for a [role] interview. Here's my work history: [brief summary].
Generate 5 STAR stories covering: leadership, problem-solving, teamwork, handling failure, and
initiative. For each, give me: Situation (2 sentences), Task (1 sentence), Action (3-4 specific
steps I took), Result (quantified outcome). Make them feel authentic, not rehearsed.
```

### 10. Behavioral Question Practice
```
I'm interviewing for a [role] at [company type]. Give me 10 likely behavioral interview
questions with: the question, what the interviewer is really assessing, a framework for
answering (including what NOT to say), and a sample strong answer. Cover: teamwork, conflict,
leadership, failure, time management, and adaptability.
```

### 11. Technical Interview Prep
```
I'm preparing for a technical interview for [role]. The job requires [key technical skills].
Create a study plan covering: core concepts I'll be tested on (ranked by likelihood),
practice problems for each area, common trick questions and how to approach them, and
questions I should ask the interviewer to show depth of knowledge.
```

## Networking & LinkedIn

### 12. LinkedIn Profile Optimizer
```
Optimize my LinkedIn profile for [target role]. Current headline: [your headline]. Summary:
[your summary]. Rewrite my headline using the formula: [Target Role] | [Key Skill] | [Value
Proposition]. Rewrite my summary: start with a hook, highlight 3 key achievements with
numbers, mention tools/technologies, end with what I'm looking for. Use keywords from this
job description: [paste JD].
```

### 13. Cold Outreach Message
```
Write a LinkedIn message to [person's role] at [company]. I want to [learn about the company /
ask about a role / request an informational interview]. My background: [1 sentence]. Keep it
under 300 characters. Be genuine, not salesy. Don't start with "I hope this message finds
you well." Find a natural hook based on [something specific about them/their company].
```

## Salary Negotiation

### 14. Salary Research Brief
```
What is the realistic salary range for a [role title] with [X years] experience in [location]?
Break down by: base salary, bonus potential, equity/stock, benefits value. Include how remote
vs on-site affects compensation, and which companies are known to pay above market.
```

### 15. Negotiation Script
```
I received a job offer for [role] at [company]: [salary offered], [other terms]. The market
rate is [your research]. I want to negotiate to [target]. Write me a negotiation script that:
expresses enthusiasm first, uses data (not emotions) to justify the ask, includes 2-3
fallback positions if they can't meet salary (signing bonus, extra PTO, remote flexibility,
review timeline), and ends with a collaborative tone. Also tell me: when to negotiate (timing),
what to never say, and how to handle "this is our final offer."
```
